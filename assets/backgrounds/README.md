# Background Images

Place Victorian-era background scenes here:

| Filename | Scene | Description |
|----------|-------|-------------|
| `drawing_room.jpg` | Main scene | Victorian drawing room/library interior |

## Finding Suitable Images

Search for public domain Victorian interiors:
- Wikimedia Commons: "Victorian drawing room interior"
- Library of Congress Digital Collections
- Historic house museum collections

## Image Requirements

- Format: JPG or PNG
- Recommended size: 1024x700 pixels or larger
- Style: Victorian era interior (1880s)
- License: Public domain or CC0 preferred

The game will render a procedural background if no image is found.
